Research Interests

I have and am currently working on realistic appearance modelling, 3D scene synthesis, and human modelling using deep learning. Representative papers are highlighted.

Neural Appearance Model for Cloth Rendering
Guan Yu Soh, Zahra Montazeri,
Eurographics Symposium on Rendering, 2024
project page / video / arXiv / code

This paper introduced a novel neural framework for modeling the light scattering of a twisted yarn by employing techniques from neural rendering. We demonstrate that our model is able to run up to 23 times faster while using less memory proportional to the number of fibers aggregated when compared to previous fiber-based methods.
